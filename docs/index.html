Teljoy source code is available on github at https://github.com/andreww5au/teljoy For any further information about Teljoy, contact Dr Andrew Williams (Andrew.Williams@Curtin.edu.au, or andrew@longtable.org).

The following files make up the actual telescope controller:

  • teljoy.py: Main executable, run this to start the telescope controller.
  • controller.py: Low level USB controller interface library from Bit Plantation.
  • correct.py: Time and astrometry class definitions.
  • detevent.py: Handles most of the high level telescope control.
  • digio.py: Module to handle digital IO for hand paddle inputs, limit switches and dome relays.
  • globals.py: constants, utility functions, and a safety interlock class. Imported by most other modules.
  • handpaddles.py: Code to handle the hand paddle buttons (N/S/E/W and speed switches).
  • motion.py: Motion control library, handling velocity ramping, tracking and slewing.
  • nzdome.py: Dome control for New Zealand system at Mt John.
  • pdome.py: Dome control for Perth system at Perth Observatory.
  • pyephem.py: Wrapper with position classes for the 'Ephem' library.
  • sqlint.py: Functions to read from and write to the local MySQL database.
  • tjserver.py: Module to run a Pyro4 proxy server making teljoy functions available via RPC over the network.
  • usbcon.py: Low level motion control driver, also handles limit switch sensing.
  • utils.py: Utility functions, almost all meant to be called manually by the user at the command line.
  • weather.py: Module to read current weather conditions from the weather server. Not used in New Zealand.

    And these files are provided to allow Teljoy to be controller via RPC over the network:

  • tjclient.py: Pyro4 client interface library, used by the clients below.
  • tjguide.py: Turns the autoguider on or off.
  • tjjump.py: Jumps the telescope to a new position, given RA/DEC/EPOCH or an object name.
  • tjoffset.py: Moves the telescope by a small offset in arcseconds E/W and N/S.
  • tjpos.py: Prints a machine-readable currentt position and status summary to standard output.
  • tjreset.py: Resets the current telescope position, given RA/DEC/EPOCH or an object name.
